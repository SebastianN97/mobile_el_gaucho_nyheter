## Getting started

### Dependencies

- Yarn
- React
- Cypress
- Axios
- Semantic-ui-react
- Image-uploading
- React-redux

### Necessary steps:

```
$ yarn install
```

### Start Cypress:

```
$ yarn cy:open
```

### Compile and deploy the React application locally:

```
$ yarn start
```
